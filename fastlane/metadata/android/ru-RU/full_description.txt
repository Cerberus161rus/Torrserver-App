Торрент-клиент.

Программа для загрузки торрентов по HTTP протоколу.
Подключение к торренту идет локально. <b>Если ваш провайдер блокирует торренты — будьте осторожны.</b>
Кэш по умолчанию хранится в оперативной памяти, поэтому не портится внутренняя флеш-память устройства.
Программа состоит из двух частей: клиент и сервер. Сервер может быть как локальным, так и располагаться в Интернете.
После первого запуска заходим в «Боковое меню» → «Обновление» → «Установить / Обновить сервер» для загрузки сервера из сети.
